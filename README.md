# Installation

Run the following to install the needed packages with pip:

`pip install -r requirements.txt`

Then also make sure Tkinter is installed:

For Linux (ubuntu):

`apt-get install python3-tk`

For Windows:
Should be installed in the python setup process.

For Mac:

`brew install python-tk`

# Usage

This repo includes code for both the client (computer doing the analysis) and the imager (the Pi).
They can be started from the `src/run.py` script.
